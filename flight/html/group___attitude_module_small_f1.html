Detailed Description

Minimal code for attitude estimation with integrated sensors.

This is a minimal implementation of the Complementary filter which integrates in the code to fetch the sensor values for Accels and Gyros and store them, and then updates a Complementary filter to estimate AttitudeActual and sets that.
