Detailed Description

Input RSSI PWM.

Driver for FrSky RSSI, which is a PWM signal with a frequency of about 110kHz. The driver uses two capture-compare channels of the timer to measure the PWM pulse width. In contrast to the normal PWM driver, this procedure does not require any interrupts.
