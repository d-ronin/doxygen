#define MAX_QUEUE_SIZE   4

Input object: FlightStatus Input object: PathDesired Input object: PositionActual Output object: StabilizationDesired

This module will periodically update the value of the StabilizationDesired object based on PathDesired and PositionActual when the Flight Mode selected in FlightStatus is supported by this module. Otherwise another module (e.g. ManualControlCommand) is expected to be writing to StabilizationDesired.
