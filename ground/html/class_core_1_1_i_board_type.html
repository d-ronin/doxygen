Detailed Description

An IBoardType object defines an autopilot or more generally a hardware device, that is supported by the GCS. It provides basic information to the GCS to detect and use this board type.

Note: at present (2012), the GCS only supports autopilots, and assumes they will talk with UAVTalk. Further down the line, GCS will be able to support additional protocols, as well as other device types (beacons, other).
